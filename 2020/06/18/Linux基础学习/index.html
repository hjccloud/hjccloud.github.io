Linux是基于Unix的

Linux是一种自由和开放源码的操作系统，存在着许多不同的Linux版本，但它们都使用了Linux内核。Linux可安装在各种计算机硬件设备中，比如手机、平板电脑、路由器、台式计算机

4、浏览文件

  • cat 由第一行开始显示文件内容
  • tac 从最后一行开始显示，可以看出 tac 是 cat 的倒着写！
  • nl 显示的时候，顺道输出行号！
  • more 一页一页的显示文件内容
  • less 与 more 类似，但是比 more 更好的是，他可以使用PgUp、PgDn翻页，
  • head 只看头几行
  • tail 只看尾巴几行
